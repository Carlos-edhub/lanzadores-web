Lanzador de CANVA y Gmail

Les presento un Script sencillo en Python para abrir Canva o Gmail directamente desde la terminal de Windows.

Características
- Abrimos Canva o Gmail instantáneamente con el comando 'canva' o 'gmail'.
- Ligero, práctico y fácil de instalar o copiar.
- Es compatible con PowerShell y Bash.

Método de instalación Windows

1. Clonamos el repositorio:
'''powershell git clone [https://github.com/TU_USUARIO/nombre-de-tu-repositorio.git](https://github.com/TU_USUARIO/nombre-de-tu-repositorio.git)

cd nombre-de-tu-repositorio

2. En caso de que sea la primera vez que vas a usar Scripts de PowerShell, debemos ejecutar el siguiente comando como Administrador:

Desde PowerShell:

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

3. Ejecutamos el instalador con el siguiente comando: 
./instalacion.ps1



- Reinicia la terminal para activar los nuevos comandos y verificar que todo funciona bien. En cuanto lo vuelvas a abrir puedes escribir 'canva' o 'gmail' para confirmar que todo funcione.
